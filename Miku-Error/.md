

### Cara replace bootanimation yang benar

Misal file baru kamu ada di:

```text
C:\Users\domathid\Downloads\bootanimation.zip
```

Jalankan ini:

```bat
adb push "C:\Users\domathid\Downloads\bootanimation.zip" /sdcard/Download/bootanimation-new.zip
```

Lalu replace ke module Magisk:

```bat
adb shell su -c "cp /sdcard/Download/bootanimation-new.zip /data/adb/modules/oshi_boot/system/product/media/bootanimation.zip"
```

```bat
adb shell su -c "cp /sdcard/Download/bootanimation-new.zip /data/adb/modules/oshi_boot/system/media/bootanimation.zip"
```

Set permission:

```bat
adb shell su -c "chmod 644 /data/adb/modules/oshi_boot/system/product/media/bootanimation.zip"
```

```bat
adb shell su -c "chmod 644 /data/adb/modules/oshi_boot/system/media/bootanimation.zip"
```

Reboot:

```bat
adb reboot
```

### Kalau file baru namanya beda

Misal namanya `Ai_bootanimation.zip` atau ada spasi, paling gampang ketik:

```bat
adb push 
```

lalu **drag file zip ke CMD**, terus tambahkan:

```bat
 /sdcard/Download/bootanimation-new.zip
```

Hasilnya kira-kira:

```bat
adb push "C:\Users\domathid\Downloads\Ai Bootanimation.zip" /sdcard/Download/bootanimation-new.zip
```

Intinya: **jangan ketik `/data/adb/...` langsung di CMD**. Itu harus selalu dibungkus `adb shell su -c "command"`.
